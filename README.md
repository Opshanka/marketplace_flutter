# MO Marketplace - Modern Flutter E-commerce App

A beautifully designed, fully functional marketplace application built with Flutter featuring dark mode, smooth animations, minimalistic design and modern UI/UX principles.

Features
- **Modern Home Screen** with search, categories, and featured products
- **Advanced Search** with real-time filtering and suggestions
- **Product Listing** with sorting and filtering options
- **Product Details** with hero animations
- **Category Navigation** with visual icons
- **Quick Actions** - Add to cart, favorites
- **Dark Mode** - Full theme switching support
- **Pull to Refresh** - Update product listings
- **Image Caching** - Better performance
- **Dynamic Product Display** - Shuffled product order

Tech Stack
- **Framework:** Flutter 3.35.4
- **Language:** Dart
- **State Management:** Provider
- **Fonts:** Google Fonts (Inter)
- **Image Caching:** cached_network_image
- **Loading Effects:** Shimmer
- **Architecture:** Clean architecture with separation of concerns
